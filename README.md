# ERC3643 Token Deployment Study (T-REX)

This repository is a personal learning and experimentation space focused on the **ERC-3643 token standard** using Tokeny’s [T-REX implementation](https://github.com/TokenySolutions/T-REX). It contains the original smart contracts and proxy architecture, alongside my custom **Hardhat deployment scripts** for testing on Ethereum testnets.

---

## 📌 About ERC-3643

**ERC-3643** (also known as T-REX) is a security token standard that brings compliance to Ethereum tokens. It supports identity-based whitelisting, transfer restrictions, and modular compliance rules — making it ideal for regulated asset tokenization.
